What is Big O Notation?

Big O notation is a mathematical notation used to describe the performance or complexity of an algorithm. It helps us understand how an algorithm scales with the size of the input.

  • O(1) - Constant Time
  • O(log n) - Logarithmic Time
  • O(n) - Linear Time
  • O(n log n) - Log Linear Time
  • O(n²) - Quadratic Time
  • O(2ⁿ) - Exponential Time
  • O(n!) - Factorial Time
